## AlternativeTo — ToolBox Image

**Title:** ToolBox Image — Privacy-First Image Processing Tools

**Description:**
ToolBox Image is a free, browser-based image processing toolkit that respects your privacy. Compress, resize, crop, rotate, convert, watermark, create passport photos, and view metadata — all without uploading your files to any server. Everything runs on your device using the HTML5 Canvas API.

**Features:**
- 100% free, no sign-up, no watermarks
- All processing happens locally (your files never leave your device)
- Supports JPEG, PNG, WebP, AVIF, GIF, HEIC (input)
- Batch processing (up to 200 images at once)
- Available in 13 languages
- No file size limits (limited by device memory)

**Key tools:** Image Compressor, Format Converter, Image Resizer, Image Cropper, Image Rotator, Passport Photo Maker, Metadata Viewer, Image Watermark
