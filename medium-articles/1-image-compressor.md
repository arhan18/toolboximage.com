## What Is Image Compression?

Image compression is the process of reducing the file size of an image without noticeably degrading its visual quality. It works by removing redundant data that the human eye barely perceives.

There are two types:
- **Lossless compression** — reduces file size without changing a single pixel. Perfect for diagrams, screenshots, and text-heavy images.
- **Lossy compression** — achieves much smaller file sizes by discarding "unimportant" visual data. Ideal for photographs where 100% pixel-perfection isn't needed.
